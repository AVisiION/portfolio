# Portfolio — Adarsh Sarangi

Personal developer portfolio built with vanilla HTML, CSS, and JavaScript.

**Live site:** https://avisiion.github.io/portfolio

---

## About

A single-page portfolio showcasing my work as an IoT and full-stack developer. Features a 3D interactive background, scroll-triggered animations, and a Bento-grid project layout.

---

## Tech

- HTML5, CSS3, Vanilla JavaScript
- [Three.js](https://threejs.org/) — 3D background
- [GSAP](https://gsap.com/) + ScrollTrigger — animations
- Hosted on GitHub Pages

---

portfolio/

## Local Development

No build step required. Open `index.html` directly in a browser, or use VS Code Live Server.
